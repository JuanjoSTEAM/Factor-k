## 📋 RESUMEN EJECUTIVO

**Factor K** es una plataforma de aprendizaje gamificada donde:
- ✅ Cada estudiante crea un perfil épico personalizado (animal, superpoder, equipo)
- ✅ 8 rutas temáticas (120-160 cursos) adaptadas por edad
- ✅ Todos los contenidos = 100% juegos interactivos
- ✅ Temas visuales automáticos según equipo/deporte favorito
- ✅ Sistema escuela + profesor + padre integrado

**Sincronización:**

## 📊 DATOS ALMACENADOS (StudentProfile)

```typescript
{
  // Básicos
  id: string
  name: string
  dateOfBirth: Date
  age: number
  gender: "niño" | "niña"
  ageGroup: "3-5" | "6-8" | "9-11" | "12-14" | "15+"
  contentLevel: "PrimerosPasos" | "Explorador" | "Aventurero" | "Emprendedor" | "Experto"
  
  // Perfil Aventurero
  heroProfile: {
    animal: string
    superpower: string
    place: string
    adventurerClass: string
    classEmoji: string
  }
  
  // Perfil Deportivo
  sportProfile: {
    sport: string
    team?: string
    teamColors: { primary: string, secondary: string }
    teamShield?: string
    favoriteColor: string
  }
  
  // Tema Visual
  appliedTheme: {
    primaryBg: string
    secondaryBg: string
    accentColor: string
    headerImage?: string
    dashboardGradient: string
  }
  
  // Progreso (por cada ruta)
  routeProgress: {
    [routeId]: {
      currentLevel: number
      currentTheme: number
      totalPoints: number
      badges: string[]
    }
  }
  
  // Score Global
  scoreSystem: {
    totalPoints: number
    streakDays: number
  }
  
  // Escuela
  schoolId?: string
  groupId?: string
}
```
